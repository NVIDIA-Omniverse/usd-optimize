# Citation

## Citing Scene Optimizer

If you use Scene Optimizer in a publication, please cite it using the following BibTeX entry (LaTeX):

```tex
@manual{nvidia2026sceneoptimizer,
  title        = {Scene Optimizer: Standalone {C++} Library for {USD} Scene Optimization},
  author       = {{NVIDIA Corporation}},
  year         = {2026},
  url          = {https://github.com/NVIDIA-Omniverse/scene-optimizer-core},
}
```

The [user manual](https://docs.omniverse.nvidia.com/extensions/latest/ext_scene-optimizer/user-manual.html) is the primary public documentation for the library and API. Source code is hosted on [GitHub](https://github.com/NVIDIA-Omniverse/scene-optimizer-core).

Adjust the `year` field to match the release or documentation version you relied on when writing.
